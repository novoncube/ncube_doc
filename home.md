# NCube 2.0

## Summary

NCube는 WPF App을 개발하면서 필요한 UI 컨트롤, 유틸리티성 코드, 통신 라이브러리를 포함한 라이브러리 ToolKit 입니다. Novo에서 많이 활용되는 UI 컨트롤을 고민했고, 자주 사용되는 UI 컨트롤을 모듈화하여 사용할 수 있도록 만들었습니다. NCube는 Novo에서 WPF 응용프로그램을 개발하기에 유용하기를 원합니다.
